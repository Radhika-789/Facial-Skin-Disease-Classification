# Dataset Information

## Dataset Used
**Name:** DermNet  
**URL:** https://www.kaggle.com/datasets/shubhamgoel27/dermnet  
**License:** Public (Kaggle)

## Subset Details
A balanced 7-class subset was created with 300 training and 60 test images per class.

**Selected Classes:**
1. Actinic Keratosis Basal Cell Carcinoma and other Malignant Lesions
2. Eczema Photos
3. Melanoma Skin Cancer Nevi and Moles
4. Nail Fungus and other Nail Disease
5. Psoriasis pictures Lichen Planus and related diseases
6. Tinea Ringworm Candidiasis and other Fungal Infections
7. Warts Molluscum and other Viral Infections

The dataset is downloaded automatically via `kagglehub` when running the notebook — no manual download required.
